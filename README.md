## 🎯 Sobre

Este repositório reúne todo o material produzido e coletado ao longo do curso de ADS. O objetivo é organizar o aprendizado e servir de consulta para colegas que estejam cursando as mesmas disciplinas.

## 🚀 Como usar

1. **Clone** o repositório:
   ```bash
   git clone https://github.com/RaoniLucas/Materiais-de-Estudo.git
   ```

2. **Navegue** pelas pastas de acordo com o semestre ou disciplina desejada.

3. **Consulte** os arquivos `.md` para resumos e anotações, e as pastas de código para exercícios e projetos.

---

## 🤝 Como contribuir

Contribuições são bem-vindas! Se você é colega de curso e quer adicionar material:

1. Faça um **fork** do repositório
2. Crie uma branch: `git checkout -b minha-contribuicao`
3. Adicione seu material na pasta correta
4. Abra um **Pull Request** descrevendo o que foi adicionado

> ⚠️ Por favor, não suba arquivos de provas que ainda estejam sendo aplicadas.
